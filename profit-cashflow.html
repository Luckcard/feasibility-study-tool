利润与利润分配表（单位：万元）
计算公式说明：
利润总额 = 营业收入 + 补贴收入 - 税金及附加 - 总成本费用
应纳税所得额 = 利润总额 - 弥补以前年度亏损
所得税 = 应纳税所得额 × 所得税率（当应纳税所得额＞0时）
净利润 = 利润总额 - 所得税
可供分配利润 = 净利润 + 期初未分配利润
法定盈余公积金 = 净利润 × 10%（当净利润＞0时）
项目投资现金流量表（单位：万元）
计算公式说明：
所得税前净现金流量 = 现金流入 - 现金流出（不含调整所得税）
调整所得税 = 息税前利润(EBIT) × 所得税率
息税前利润 = 营业收入 - 税金及附加 - 经营成本 - 折旧 - 摊销
所得税后净现金流量 = 所得税前净现金流量 - 调整所得税
累计净现金流量 = 各年净现金流量累加
项目资本金现金流量表（单位：万元）
计算公式说明：
净现金流量 = 现金流入 - 现金流出
现金流出含：项目资本金投入、借款本金偿还、借款利息支付、经营成本、税金及附加、所得税
本表从投资者角度考察项目资本金的盈利能力。
财务计划现金流量表（单位：万元）
计算公式说明：
经营活动净现金流量 = 营业收入 - 经营成本 - 税金及附加 - 所得税
投资活动净现金流量 = -建设投资 - 流动资金 - 维持运营投资 + 回收资产余值
筹资活动净现金流量 = 资本金 + 借款取得 - 借款本金偿还 - 借款利息支付 - 股利分配
净现金流量 = 经营活动 + 投资活动 + 筹资活动净现金流量
累计盈余资金 = 各年净现金流量累加（反映资金充裕程度）
